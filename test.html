Instruction:


This test is for measuring the reaction time. It is called reaction time test. This is a computer based test. It is used to test human movement perception, specially speed and time perception. This test is performed to test your quick response time.


How to work:

  • Press L if no color appear on screen.
  • On the monitor, there will be a traffic signal containing any one of 3 lights: Red, green or yellow.
  • On the keyboard you have to keep your finger on blue key (L) all the time.
  • Blue key represents neutral mode. You have to use only one finger all the time.

When a light blinks on the monitor, see the color of the light. Release the blue key, press the key which corresponds to the light ('R' key for Red/'G' key for Green/'Y' key for Yellow) and come back to blue one immediately. Do it as fact as possible . The test goes on for about 1-2 minutes. Don't press wrong color key at any cost. Although nobody is sure, but it is said that 100% accuracy is required in this test.
